How to run?

1- Admin App:
--Install xampp
--Place Admin folder here present in php scripts 
in xampp/htdocs
--Open phpMyadmin on browser through xampp
--Run DB schema script in MYSQL (phpMyadmin)
-- Open this AdminAppCode\app\src\main\java\com\example\myfyp\URLs.java change ip address(ipconfig on cmd) 
according to your internet
-- Mob and Laptop should be connected to same internet
--Change built.gradle according to your android version

2- Data Capturing App:
--Change built.gradle according to your android version
